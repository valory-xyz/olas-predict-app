# Olas Predict App

This repo contains the UI for the [Olas Predict](https://olas.network/agent-economies/predict)

## Tech Stack
- NodeJS (20.16 LTS)
- ReactJS
- NextJS
- TypeScript
- AntD
- Styled components


### Installation:

`yarn`

### Start the app

`yarn dev`

### Build the app

`yarn run build`
